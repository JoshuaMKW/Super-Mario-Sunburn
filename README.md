# Super Mario Sunburn

Welcome to the home of **Super Mario Sunburn's** source code!

## Requirements

- Visual Studio 2022

## Compile

This code compiles into the Kuribo module that drives Super Mario Sunburn. To compile it, select a build configuration and then build.
